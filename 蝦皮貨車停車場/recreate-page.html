🚚 蝦皮貨車位置查詢 - 重建列印頁面

將 about:blank 列印頁面的 HTML 貼上，即可在此重新顯示並使用擴充套件功能

步驟一：貼上 HTML 原始碼

📋 如何複製列印頁面的 HTML：

  1. 在 about:blank 列印頁面按下 F12 開啟開發者工具
  2. 切換到 Elements 或 元素 標籤
  3. 在 <html> 標籤上按右鍵
  4. 選擇 Copy → Copy outerHTML 或 複製 → 複製 outerHTML
  5. 將複製的內容貼到下方文字框
💡 按鈕說明：
• 🔄 重建頁面：標準模式，會進行最小化包裝
• 💯 完全匹配模式：完全不修改原始HTML，保持100%一致的格式
• 🚚 掃描貨車位置：在重建後的頁面中查找並標記貨車位置
