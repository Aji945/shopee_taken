🚚 蝦皮貨車位置查詢 - 直接重建工具

創建一個與 about:blank 頁面完全相同的新頁面，解決格式差異問題

步驟一：貼上 HTML 原始碼

📋 如何複製列印頁面的 HTML：

  1. 在 about:blank 列印頁面按下 F12 開啟開發者工具
  2. 切換到 Elements 或 元素 標籤
  3. 在 <html> 標籤上按右鍵
  4. 選擇 Copy → Copy outerHTML 或 複製 → 複製 outerHTML
  5. 將複製的內容貼到下方文字框

🎯 全新解決方案

直接重建模式：創建一個全新的瀏覽器頁面，而不是使用iframe

這樣可以確保與原始頁面100%完全相同的渲染效果

重建後的頁面會在新分頁中開啟，格式應該與原始頁面完全一致

步驟二：使用重建的頁面

📖 使用說明：

  1. 點擊 🎯 直接重建（新分頁） 按鈕
  2. 新分頁會開啟，顯示與原始頁面完全相同的內容
  3. 在新分頁中使用 蝦皮貨車停車場 擴充套件
  4. 擴充套件會自動偵測並標記貨車位置
  5. 或者下載HTML檔案，在本地開啟使用
