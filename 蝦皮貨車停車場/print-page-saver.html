🚚 蝦皮貨車位置查詢 - 列印頁面保存工具

由於 about:blank 頁面的限制，這裡提供幾種不同的解決方案

🎯 方法一：使用鍵盤快捷鍵複製HTML（推薦）

步驟 1：開啟開發者工具
在 about:blank 列印頁面按下 F12 或 Ctrl+Shift+I
步驟 2：切換到 Elements 標籤
點擊開發者工具中的 "Elements" 或"元素" 標籤
步驟 3：選擇 HTML 根元素
在 Elements 面板中找到 <html> 標籤並點擊選中
步驟 4：使用鍵盤快捷鍵複製
按下 Ctrl+C 複製整個HTML結構
或者在選中的 <html> 標籤上按右鍵選擇 "Copy" → "Copy outerHTML"
步驟 5：貼到重建工具
將複製的HTML貼到工具中並重建

💾 方法二：直接保存頁面

步驟 1：保存頁面
在 about:blank 頁面按下 Ctrl+S
步驟 2：選擇保存位置
選擇保存類型為 "網頁，完整" 或 "Web Page, Complete"
步驟 3：開啟保存的檔案
在檔案總管中開啟保存的 .html 檔案，擴充套件就可以正常運作了
✅ 優點： 最簡單的方法，保存的檔案格式完全相同

🖨️ 方法三：列印到PDF

步驟 1：列印
按 Ctrl+P 開啟列印對話框
步驟 2：另存為PDF
選擇 "另存為PDF" 或 "Save as PDF"
步驟 3：手動輸入
查看PDF內容，手動輸入到檢貨系統中
⚠️ 缺點： 需要手動輸入，無法自動標記貨車位置

📷 方法四：截圖識別

步驟 1：截圖
使用 Win+Shift+S 截取頁面
步驟 2：文字識別
使用 OCR 工具識別文字內容
步驟 3：手動處理
將識別的文字輸入到檢貨系統
⚠️ 缺點： 準確度可能不高，需要人工校對

🎯 推薦的完整流程

最佳解決方案：
  1. 首先嘗試 方法二（Ctrl+S 保存頁面） - 最簡單且有效
  2. 如果保存失敗，使用 方法一（F12 + 複製HTML）
  3. 將保存的檔案或重建的頁面用蝦皮貨車停車場擴充套件處理
  4. 擴充套件會自動標記所有商品的貨車位置
